# CORS 总结
报错’access-control-allow-origin‘的字段   跨域
后台日志会显示已经返回数据实际上已经被浏览器拦截
IP加入cors跨域白名单
后台如果当前请求的origin或者refer不在白名单，就直接返回404或者空值，减少服务器压力
